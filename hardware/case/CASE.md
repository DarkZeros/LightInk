# Case Design 

The case started from a reference case from: https://www.printables.com/model/502517-watchy-case-with-low-profile-buttons/collections
Thanks for that reference design!

I took it and edited on top:
  - Removed the screw holes
  - Shifted the display UP in order to have a even space in the border around display
    - This required removing the watchy PCB arches on top/bottom
  - Make a new square hole for the solar panel
  - Removed the buttons, instead add M1 holes for screws
  - Make more space inside the case by reducing the supports and the reducing size
  - Many other small changes

# Important things

The case needs to be UV resistent. If the material is not, then will quickly turn YELLOW.
I used SLA resin and Nylon, and those did not degrade, but other materials probably will.
